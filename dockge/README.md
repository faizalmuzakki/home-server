# Dockge

Docker Compose management UI by the creator of Uptime Kuma.

## Features

- Start/stop/restart stacks from web UI
- Real-time log viewing
- Edit compose files in browser
- Stack status monitoring

## Access

- **Local**: http://server-ip:5001
- **Via Traefik**: https://dockge.yourdomain.com

## First Run

1. Open the web UI
2. Create an admin account
3. All stacks in `/home-server` will be automatically detected

## Environment Variables

| Variable | Description | Default |
|----------|-------------|---------|
| `DOMAIN` | Your domain for Traefik routing | - |
